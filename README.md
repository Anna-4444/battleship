#  Battleship

A fully interactive web-based implementation of the classic Battleship game — built with **JavaScript, HTML, and CSS**, bundled using **Webpack**.  
This project was created as part of [The Odin Project](https://www.theodinproject.com/) curriculum.

---

## Live Preview
[Click here:](https://anna-4444.github.io/battleship/)

---

##  Features

- **Single Player Mode:** Play against a computer opponent that makes random attacks.  
- **Two Player Mode:** Challenge a friend locally by taking turns on the same device.  
- **Drag and Drop Ship Placement:** Easily position your fleet on the grid before the game begins.  
- **Interactive Boards:** Click to attack enemy coordinates and see visual hit/miss feedback with sound effects.  
- **Turn-Based Gameplay:** Smooth transitions between turns, including pass screens for 2-player mode.  
- **Game Over Detection:** The game ends automatically when all ships are sunk, with a replay option.  

---

##  Future Improvements

- **Improve computer intelligence:** Upgrade the computer opponent to make intelligent attacks after scoring a hit (target adjacent cells).   
- **Mobile Optimization:** Improve drag and drop and layout for touch-based devices.  

---

##  Technologies Used

- **HTML5** – Structure and layout of the app  
- **CSS3** – Styling and responsive grid design  
- **JavaScript (ES6 Modules)** – Core game logic and DOM interaction  
- **Webpack** – Module bundling and development server  
- **npm** – Dependency and build management
